三角形最小路径和
2020-09-24|leetcode

给定一个三角形，找出自顶向下的最小路径和。每一步只能移动到下一行中相邻的结点上。

相邻的结点 在这里指的是 下标 与 上一层结点下标 相同或者等于 上一层结点下标 + 1 的两个结点。

例如，给定三角形：

[
[2],
[3,4],
[6,5,7],
[4,1,8,3]
]
自顶向下的最小路径和为 11（即，2 + 3 + 5 + 1 = 11）。

说明：

如果你可以只使用 O(n) 的额外空间（n 为三角形的总行数）来解决这个问题，那么你的算法会很加分。

备注：相邻的结点 在这里指的是 下标 与 上一层结点下标 相同或者等于 上一层结点下标 + 1 的两个结点。

这句话很有歧义：应该是 相邻的结点 在这里指的是 下标 与 下一层结点下标 相同或者等于 下一层结点下标 + 1 的两个结点。

在这种情况下：dp[i] = min(dp[i], dp[i -1]) + value

因为dp的坐标会受到前一个坐标的影响，所以这里一定要 for (int i = list.size() - 1; i >= 0; i–)倒序操作
